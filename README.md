# 몸과 창작 공부 · 루틴 체크 앱

나의 일상 루틴을 기록하고  
몸, 창작, 공부의 균형을 점검하기 위해 만든 개인 웹앱입니다.

## 🌿 주요 기능

- 하루 루틴 체크 (완료 / 누적 +1)
- 하루 목표 달성 여부 표시
- 최근 30일 / 6개월 누적 통계
- 모바일에서도 사용 가능
- GitHub Pages로 배포된 PWA 웹앱

## 🧘 나의 기본 루틴

- 달리기 4km (이틀에 1번)
- 배변 1회
- 기타연습 20분
- 독서 및 요약
- 어반스케치 (주 1회)
- 물마시기
- 50분 공부 후 스트레칭 + 눈 휴식
- 글쓰기 (2주 1회)

## 🚀 사용 방법

웹 주소:
