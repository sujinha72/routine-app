# 몸과 창작 공부 · 루틴 체크

작은 실행의 누적이 삶을 만든다고 믿습니다.

이 앱은  
몸 · 창작 · 공부의 균형을 스스로 점검하기 위해  
직접 만들었습니다.

기술보다 중요한 것은 방향이고,  
기록보다 중요한 것은 리듬입니다.

---

## 🌿 왜 만들었는가

목표는 거창하지 않습니다.

- 이틀에 한 번 달리기
- 하루 한 번 배변 체크
- 기타 20분
- 독서 후 한 줄 요약
- 어반스케치
- 물 마시기
- 50분마다 몸 일으키기
- 2주에 한 번 글쓰기

거대한 변화 대신  
작은 실행을 매일 쌓아가는 구조를 만들고 싶었습니다.

---

## ⚙ 기능

- 하루 루틴 체크 (완료 / 누적 +1)
- 하루 목표 달성률 표시
- 최근 30일 / 6개월 누적 통계
- 모바일 홈 화면 설치 가능 (PWA)
- 데이터는 개인 브라우저에 저장

---

## 🧘 이 앱의 철학

나는 장 속 미생물과 공생하는 하나의 소우주입니다.  
지구 환경을 돌보듯  
몸과 리듬을 돌보고 싶었습니다.

기록은 통제가 아니라  
리듬을 확인하는 도구입니다.

완벽이 아니라 지속.  
속도가 아니라 방향.
